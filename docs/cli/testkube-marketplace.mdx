## testkube marketplace

Browse and install TestWorkflows from the Testkube Marketplace

### Synopsis

Browse, inspect, and install TestWorkflows from the Testkube Marketplace
(https://github.com/kubeshop/testkube-marketplace) without opening the dashboard.

```
testkube marketplace [flags]
```
