## testkube update testworkflowexecution

Update test workflow execution tags

### Synopsis

Update test workflow execution tags. Use --tag key=value to set tags. Replaces all existing tags. Omit --tag to clear all tags.

```
testkube update testworkflowexecution <executionID> [flags]
```

### Options

```
  -h, --help              help for testworkflowexecution
      --tag stringArray   tag in key=value format (can be specified multiple times, omit to clear all tags)
```
