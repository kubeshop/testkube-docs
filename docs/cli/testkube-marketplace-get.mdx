## testkube marketplace get

Show detailed information about a marketplace TestWorkflow

### Synopsis

Displays the catalog entry, configurable parameters, and optionally the raw YAML
and readme content for a marketplace TestWorkflow.

```
testkube marketplace get <name> [flags]
```

### Options

```
  -h, --help          help for get
      --show-readme   include the workflow readme in the output
      --show-yaml     include the raw workflow YAML in the output
```
