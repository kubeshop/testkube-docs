## testkube marketplace list

List TestWorkflows available in the marketplace

### Synopsis

Fetches the marketplace catalog and lists available TestWorkflows. Use --category and --component to filter, or --output to switch formats.

```
testkube marketplace list [flags]
```

### Options

```
      --category string    filter by category (e.g. databases, networking)
      --component string   filter by component (e.g. redis, postgresql)
  -h, --help               help for list
```
