# Testkube Examples and Guides

This section gives examples on how to create Workflows for the most common testing tools - if you're missing
something please let us know on our [Slack Channel](https://bit.ly/testkube-slack)!

## Load & Performance Testing

Tools for load testing, stress testing, and performance benchmarking:

- K6
  - [Basic](k6-basic) - Shows how to run a simple k6 test.
  - [Distributed](k6-distributed) - Shows how to distribute a K6 test across multiple nodes.
  - [Testkube K6 Introduction](https://testkube.io/blog/load-testing-in-kubernetes-with-k6-and-testkube) (external)
  - [Testkube vs K6 Operator](https://testkube.io/learn/comparing-the-k6-operator-vs-testkube-for-load-testing) (external)
  - [Generating Massive Load](https://testkube.io/learn/a-guide-to-scalable-and-heavy-load-testing-with-k6-and-testkube) (external)
- JMeter
  - [Basic](jmeter-basic) - Shows how to run a simple JMeter test.
  - [Distributed](jmeter-distributed) - Shows how to run a JMeter test with remote workers.
  - [Plugins](jmeter-plugins) - Shows how to use JMeter plugins.
- [Gatling](gatling-basic) - Shows how to run a basic Gatling load-test.
- Locust
  - [Basic](locust-basic) - Shows how to run a simple Locust test.
  - [Distributed](locust-distributed) - Shows how to run a Locust test with remote workers.
- [Artillery](artillery-basic) - Shows how to run a basic Artillery test.

## End-to-End & Browser Testing

Tools for UI testing, browser automation, and end-to-end testing:

- Cypress
  - [Basic](cypress-basic) - Shows how to run a Cypress test.
  - [Sharded](cypress-sharded) - Shows how to distribute a Cypress test across multiple nodes for sharded test execution.
  - [Scaling Cypress Tests](https://testkube.io/learn/scaling-cypress-tests-parallelise-your-end-to-end-tests-with-testkube) (external)
- Playwright
  - [Basic](playwright-basic) - Shows how to run a Playwright test.
  - [Sharded](playwright-sharded) - Shows how to distribute a Playwright test across multiple nodes for sharded test execution.
  - [Rerun Failed Tests](playwright-rerun) - Shows how to rerun failed Playwright tests.
- Selenium
  - [Basic](selenium-basic) - Shows how to run a Selenium test with Maven against the Chrome WebDriver Engine.
  - [Advanced](selenium-advanced) - Shows how to run a Selenium test with Maven and 3 browser engines; chrome, edge and firefox.
  - [Functional Testing with Selenium](https://testkube.io/learn/functional-testing-with-selenium-in-testkube) (external)
  - [Selenium Grid with Testkube](https://testkube.io/learn/faster-smarter-selenium-testing-with-testkube-on-kubernetes) (external)
- Robot Framework
  - [Basic](robot-framework-basic) - Shows how to run a basic Robot Framework test.
  - [Parallelized](robot-framework-parallel) - Shows how to run a Robot Framework test in parallel for 3 browsers.
  - [Automated Acceptance Testing](https://testkube.io/learn/automated-acceptance-testing-with-robot-framework-and-testkube) (external)
