# Failed Execution Details

A Testkube **Failed execution** is a test that has completed but did not pass all required checks.

![Failed Executions](../articles/images/testkube-dashboard-exec-fail-test.png)

After you select a test from the **Failed executions** page, you can view its details and debug issues. You can also **Run again**.

![Failed Execution Details](../articles/images/testkube-dashboard-exec-fail-test-details.png)
